HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

  • Headers max out at 6.
  • The Paragraph tag 'p' goes at the bottom of the page
  • 'strong' tag around text will make the text bold.
  • 'ul' unordered list, bullet point list
  • 'ol' ordered list, numbered list
  • 'a' anchor tag, href="URL"

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • font-family: Determines the font of a page
  • font-size: Size of the font
  • line-height: Height of the line of text
